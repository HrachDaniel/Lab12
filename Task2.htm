Команди Git

Створення репозиторію

git init [назва_проекту]

git clone [URL]

Робота з гілками

git branch

git branch -av

git checkout [назва_гілки]

git branch [нова_гілка]

git branch -d [назва_гілки]

git checkout [гілка_b]

git merge [гілка_a]

git tag [тег]

Внесення змін

git add [файл]

git add .

git commit -m "[повідомлення] "

git commit -am "[повідомлення]"

git reset [файл]

git reset --hard

Синхронізація

git fetch

git pull

git pull --rebase

git push

Спостереження за репозиторієм

git status

git diff

git diff --cached

git diff HEAD

git diff [commit1] [commit2]

git blame [файл]

git show [commit]:[файл]

git log

git log -p [файл/директорія]

Наостанок

git help [команда]
